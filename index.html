Verilog HDL学习

发表于 2020-08-10 分类于 FPGA

整数型表示方法

(1) 8’hab 表示 8bit 的十六进制数，换算成二进制是 1010_1011；
(2) 8’d171 表示 8bit 的十进制数，换算成二进制是 1010_1011；
(3) 8’o253 表示 8bit 的八进制数，换算成二进制是 1010_1011；
(4) 8’b1010_1011 表示 8bit 的二进制数，二进制就是 1010_1011。

ffmpeg转码

发表于 2020-05-10 分类于 ffmpeg

最近抓m3u8文件的ts流，每次抓到的都是几百个ts文件。

新建filelist.txt，文本内容格式：

file '1.ts'
file '2.ts'

命令行执行：

$ffmpeg -f concat -i filelist.txt output.mkv

jpg转成mkv

$ffmpeg -framerate 30 -i input%03d.jpg -codec copy output.mkv

树莓派开箱

发表于 2020-05-10 更新于 2020-10-18 分类于 raspberry

食用

在boot分区添加以下文件

  • ssh
  • wpa_supplicant.conf

ssh 开启ssh
wpa_supplicant.conf设置无线wifi

开机

树莓派自动连上wifi

通过ssh登录树莓派

用户名：pi

初始密码：raspberry

更改国内软件源

  • 添加root密码

      sudo passwd root

  • 修改用户pi的密码

      passwd pi

  • 修改国内软件源

    设置静态IP

    开启vnc

      sudo raspi-config
  5 Interfacing Options
  p3 VNC
  7 Advanced Options
  # 设置分辨率，不是默认的就行
  A5 Resulution

设置中文

    sudo raspi-config
    4 localisation Options
    I1 Change Locale
    Zh_CN.UTF-8     # 空格键选中

ADB禁用、删除软件

手机自带软件无法删除或禁用，比如手机上的自带的音乐和视频软件。

ADB下载地址： ADB

  1. 手机打开USB调试，各个手机都不同，具体的请百度；
  2. 下载并解压ADB.rar文件夹；
  3. 管理员模式运行ADBDriverInstaller.exe并安装驱动；
  4. 管理员模式打开CMD，进入ADB文件夹（cd命令）
