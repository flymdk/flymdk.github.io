树莓派开箱

发表于 2020-05-10 更新于 2020-10-18 分类于 raspberry

食用

在boot分区添加以下文件

  • ssh
  • wpa_supplicant.conf

ssh 开启ssh
wpa_supplicant.conf设置无线wifi

开机

树莓派自动连上wifi

通过ssh登录树莓派

用户名：pi

初始密码：raspberry

更改国内软件源

  • 添加root密码

      sudo passwd root

  • 修改用户pi的密码

      passwd pi
